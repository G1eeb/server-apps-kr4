# Контрольная работа №4 - Технологии разработки серверных приложений

## Описание проекта

FastAPI приложение с миграциями БД (Alembic), пользовательскими исключениями, валидацией данных и тестами (синхронными и асинхронными).

### Выполненные задания

- **9.1** - Alembic миграции для модели Product с полем description
- **10.1** - Пользовательские исключения (CustomExceptionA, CustomExceptionB)
- **10.2** - Валидация данных Pydantic + кастомный обработчик ошибок
- **11.1** - Модульные тесты (синхронные, pytest + TestClient)
- **11.2** - Асинхронные тесты (pytest-asyncio + httpx.AsyncClient + Faker)
